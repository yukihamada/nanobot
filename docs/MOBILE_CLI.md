# 📱 モバイルCLI機能ガイド

chatweb CLIにスマホフレンドリーな機能が追加されました！

## 🎮 絵文字イースターエッグ

特定の絵文字パターンを入力すると、隠しボーナスクレジットがもらえます！

### 💕 ラブパターン
- `❤️❤️❤️` - LOVE BONUS
- `💕💕💕` - LOVE BONUS

### 🎉 お祝いパターン
- `🎉🎉🎉` - CELEBRATION BONUS
- `🎊🎊🎊` - CELEBRATION BONUS
- `🎁🎁🎁` - CELEBRATION BONUS

### ✨ スターパターン
- `✨✨✨` - STAR BONUS
- `⭐⭐⭐` - STAR BONUS
- `🌟🌟🌟` - STAR BONUS

### 🎮 ゲームパターン
- `🎮🎮🎮` - GAMER BONUS
- `🎯🎯🎯` - GAMER BONUS

### 記号パターン
- `!!!` - ENERGY BONUS
- `???` -
- `...` - THINKING BONUS

### 数字パターン
- `123`, `1234`, `321` - NUMBER BONUS

### テキストパターン
- `love`, `LOVE` - SECRET BONUS
- `ありがとう❤️` - SECRET BONUS
- `すごい！！！` - SECRET BONUS

## 🎯 クイックメニュー

### メインメニュー表示
```bash
/q  または  ?
```

メニュー内容：
1. 📊 ステータス確認
2. 🎮 コナミコード実行
3. 🔗 セッション連携
4. 💬 よく使うフレーズ
5. 📋 ヘルプ表示

### 数字でクイックアクション
メッセージ入力欄で数字（1-5）を入力すると、対応するアクションが実行されます。

## 💬 よく使うフレーズ

### フレーズメニュー表示
```bash
/m
```

登録フレーズ：
1. こんにちは！
2. これについて詳しく教えて
3. わかりやすく説明して
4. 要約して
5. ありがとう！

フレーズメニューを開いた後、数字（1-5）で選択すると自動送信されます。

## ⚡ ショートカットコマンド

| コマンド | 機能 |
|---------|------|
| `/q` または `?` | クイックメニュー表示 |
| `/s` | ステータス表示 |
| `/h` | ヘルプ表示 |
| `/c` | 画面クリア |
| `/.` | 前回メッセージ再送 |
| `/m` | よく使うフレーズメニュー |
| `/konami` | コナミコード実行 |

## 💡 Tips

- 絵文字イースターエッグは1回のみ使用可能（コナミコードと同様）
- クイックメニューは数字だけで操作できるのでスマホで便利
- よく使うフレーズは自分好みにカスタマイズ可能（コード編集）
- 画面クリア（/c）で見やすさをリセット

## 🚀 起動方法

```bash
# デフォルト（Voiceモード）
chatweb

# チャットモード（モバイル機能付き）
chatweb chat
```

スマホのターミナルアプリ（Termux、iSH、Blink Shell等）で快適に使えます！
