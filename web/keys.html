from openai import OpenAI

client = OpenAI(
    api_key="cw_your_api_key_here",
    base_url="https://api.chatweb.ai/v1",
)

response = client.chat.completions.create(
    model="minimax/minimax-m2.5",  # or any model
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: "cw_your_api_key_here",
  baseURL: "https://api.chatweb.ai/v1",
});

const response = await client.chat.completions.create({
  model: "minimax/minimax-m2.5",
  messages: [{ role: "user", content: "Hello!" }],
});
console.log(response.choices[0].message.content);

chatweb.aiはOpenRouter互換のAIゲートウェイです。同じモデル名・同じSDKでそのまま切り替えられます。
Developer Modeを有効にすると会話がENAIトークン報酬に変わります。
対応モデル: GET /v1/models で一覧取得できます。 minimax/minimax-m2.5, moonshotai/kimi-k2.5, openai/o4-mini, anthropic/claude-* など100+モデルに対応。 クレジット残高が足りない場合は HTTP 402 が返ります。
