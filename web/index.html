Connect Your Channel

Add your bot to LINE or Telegram and start using AI right away

LINE

Connect your LINE Official Account to nanobot. Your bot will respond to messages automatically.

  1. 1 Go to LINE Developers Console and create a Messaging API channel
  2. 2 Copy your Channel Secret and Channel Access Token
  3. 3 Set webhook URL to: https://api.chatweb.ai/webhooks/line
  4. 4 Enable "Use webhook" and click Verify
Open LINE Developers Console

Telegram

Create a Telegram bot and connect it to nanobot. Messages are processed instantly via webhook.

  1. 1 Message @BotFather on Telegram and create a new bot
  2. 2 Copy the Bot Token from BotFather
  3. 3 Register webhook (auto-configured on deploy)
  4. 4 Send a message to your bot and get an AI reply
Open BotFather

API Quick Start

# Send a message to your agent
curl -X POST https://api.chatweb.ai/api/v1/chat \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello!", "session_id": "user-123"}'

# Response
{
  "response": "Hello! How can I help you today?",
  "session_id": "user-123"
