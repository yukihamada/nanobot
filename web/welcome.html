1 APIを呼び出す

curlコマンドでAPIを試してみましょう。YOUR_API_KEYをメールで受け取ったキーに置き換えてください。

# Send a chat message curl -X POST https://chatweb.ai/api/v1/chat \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{ "message": "Hello, chatweb.ai!", "session_id": "my-session-1" }'

2 レスポンスを確認

APIは以下の形式でJSON応答を返します。

{ "response": "Hello! How can I help you today?", "session_id": "my-session-1" }

3 使用量を確認

いつでもAPIで使用量とクレジット残高を確認できます。

# Check usage curl https://chatweb.ai/api/v1/usage \ -H "Authorization: Bearer YOUR_API_KEY"
