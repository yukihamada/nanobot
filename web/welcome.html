セッションキーでAPIを利用できます。早速使い始めましょう。

あなたのセッションキー

---
セッションキーはAPIの認証に使用します

1 APIを呼び出す

curlコマンドでAPIを試してみましょう。

# Send a chat message curl -X POST https://chatweb.ai/api/v1/chat \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_SESSION_KEY" \ -d '{ "message": "Hello, chatweb.ai!", "session_id": "YOUR_SESSION_KEY" }'

2 レスポンスを確認

APIは以下の形式でJSON応答を返します。

{ "response": "Hello! How can I help you today?", "session_id": "YOUR_SESSION_KEY", "credits_used": 1, "credits_remaining": 99 }

3 クレジット残高を確認

いつでもAPIでクレジット残高を確認できます。

# Check credits and account info curl https://chatweb.ai/api/v1/auth/me \ -H "Authorization: Bearer YOUR_SESSION_KEY"
