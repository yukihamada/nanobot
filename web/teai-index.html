FAQ

よくある質問

teai.ioについてよく寄せられる質問です。

teai.io は chatweb.ai と何が違うの？ +

teai.io は日本語に特化したAIチャット＆API Gateway です。chatweb.ai と同じバックエンドを使いつつ、日本語モデル（Nemotron 9B 無料）・東京リージョン・日本語サポートに最適化しています。APIキー1つで GPT-4o・Claude・Gemini を切り替えられる点が最大の特徴です。

無料プランで何ができますか？ +

登録するだけで 100 クレジットがもらえます。Nemotron 9B（日本語特化）は無制限で無料。GPT-4o・Claude などのプレミアムモデルはクレジットを消費します。チャットUI（/chat）とAPI両方が使えます。クレジットがなくなったら Starter プランへのアップグレードで継続利用できます。

既存の OpenAI SDK からどうやって乗り換えますか？ +

たった1行の変更で完了します。base_url="https://api.teai.io/v1" に変更し、teai.io の APIキー（te_xxxx）を設定するだけです。LangChain・LlamaIndex・LiteLLM など OpenAI 互換のライブラリはすべてそのまま動作します。

入力したプロンプトはどこかに保存されますか？ +

リクエスト本文（プロンプト・レスポンス）は一切保存しません。保存するのはトークン数・モデル名・タイムスタンプなどのメタデータのみ（90日保持）。BYOKモードでは teai.io を経由せず直接プロバイダーAPIを呼び出すため、データが一切通過しません。

法人での請求書払いや領収書発行はできますか？ +

Pro 以上のプランで請求書払い・インボイス対応が可能です。月次請求書・適格請求書（インボイス）の発行に対応しています。法人契約・DPA（データ処理契約）が必要な場合は info@enablerhq.com までお問い合わせください。
